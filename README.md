Antes de comenzar, asegúrate de tener instalado:

- **Node.js** (versión 16 o superior) - [Descargar Node.js](https://nodejs.org/)
- **Git** - [Descargar Git](https://git-scm.com/downloads)
- **npm** (viene incluido con Node.js)

**Instalar las dependencias**:


npm install


### Iniciar el Frontend


npm run dev
El servidor frontend estará corriendo en: **http://localhost:5173**

**Nota**: Asegúrate de que el backend esté corriendo en el puerto 3000 para que el frontend pueda conectarse correctamente.
